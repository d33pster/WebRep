## Contains Project WebRep v6.0
## Currently configured for LINUX WINDOWS and macOS

## Prereqs ~ Python3

## GUI
1. git clone the repo ```https://github.com/d33pster/WebRep.git```
2. ```cd WebRep```
#### Windows
3. ```python prep```
4. ```python setup```
#### Linux and macOs
3. ```./prep```
4. ```./setup```
## If GUI fails... try this
#### Windows
1. ```python3 prep```
2. ```python3 webrep.py```
#### Linux and macOs
1. ```./prep```
2. ```./webrep.py```
